# Portafolio Personal — Luis López

Portafolio web personal desarrollado con HTML, CSS y JavaScript.

## Estructura del proyecto

```
portafolio-luis/
├── index.html       # Página principal
├── css/
│   └── style.css    # Estilos y diseño responsivo
├── js/
│   └── main.js      # Interacciones y animaciones
├── img/
│   └── foto.png     # Foto de perfil
└── README.md
```

## Características

- Diseño responsivo (desktop, tablet y mobile)
- Navegación fija con menú hamburguesa en mobile
- Animaciones de entrada al hacer scroll
- Secciones: Hero, Sobre mí, Habilidades, Proyectos, Experiencia, Contacto
- Sin dependencias externas ni frameworks

Solo abre `index.html` en tu navegador. No requiere servidor ni instalación.
